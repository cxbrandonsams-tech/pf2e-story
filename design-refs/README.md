# Design References

Drop screenshots and reference images here to show how you want the book to look and how the page-flip animation should behave.

Suggested naming:
- `book-style-*.png` — overall book look (cover, spread, textures)
- `page-layout-*.png` — page layouts (text placement, image framing)
- `flip-animation-*.png` / `.gif` / `.mp4` — desired page-flip feel
- `controls-*.png` — control bar style references

Nothing in this folder is used by the live site — it's just for planning.
